2.8.2

<p>This is a feature release for JForum. It includes the following improvements, amongst others:</p>

<ul>
	<li>Fixed a bug in the search page pagination</li>
	<li>Fixed a bug that caused posts that were awaiting moderation to be found through the search</li>
	<li>No longer uses an external library to detect access from a mobile device</li>
	<li>Updates jforum-custom.conf:server.info when detecting an application server version change</li>
	<li>numerous dependency updates</li>
</ul>

<p>See <a href="https://sourceforge.net/p/jforum2/wiki2/NewFeatures282/">this page</a> for a full list of what is new and changed.</p>

<p>Everyone should upgrade. A Servlet 3.1 container, like Tomcat 8 or newer, running on Java 8 or newer, is required.</p>
